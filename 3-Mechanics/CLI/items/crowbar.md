# Crowbar
*Adventuring Gear*  

- **Cost**: 2 gp
- **Weight**: 5.0 lbs.

Using a crowbar grants advantage to Strength checks where the crowbar's leverage can be applied.

*Source: Player's Handbook p. 151. Available in the SRD and the Basic Rules.*
